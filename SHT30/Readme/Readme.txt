文件夹介绍；
User :存放Main函数 一般不用操作 （因为对内部进行了封装，在其他地方即可操作）

Imliement:执行函数文件夹：存放主函数的执行函数、相关协议的功能实现函数、数据格式处理的函数

Peripheral:单片机片上外设的驱动

HardDriver:除外设外其他硬件设备的驱动

SoftDriver:与软件进行交互时的操作函数

Protocol: 协议库

Delay: 不同的延时函数

HeaderFilers:全部头文件 （应该避免这种写法，用谁调用谁比较合理）

Check:校验函数

System:系统配置
